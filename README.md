## GitHub Pages Deployment

Recommended setup:

1. Create a GitHub repository, for example `heshik-personal-site`.
2. Put the contents of this folder at the repository root.
3. Commit and push to GitHub.
4. In GitHub, go to `Settings` -> `Pages`.
5. Under `Build and deployment`, choose:
   - Source: `Deploy from a branch`
   - Branch: `main`
   - Folder: `/root`
6. Save.

GitHub will publish the site at the Pages URL shown in the settings screen.

## Optional Custom Domain

If using a custom domain later, add a `CNAME` file containing only the domain name.
